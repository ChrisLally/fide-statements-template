<AttestationVerifier />

This tool verifies:

- Statement membership in an attestation (`proof` + `statementFideId`)
- Signature validity for `ed25519`, `eip712`, or `eip191`

For `ed25519`, provide a raw 32-byte public key in `0x...` hex or base64.
