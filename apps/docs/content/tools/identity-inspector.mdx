Use the **Identity Inspector** to understand how the Fide Context Protocol resolves identifiers into canonical identities. Enter a **full Fide ID** (`did:fide:0x...`), or use **entity type + source + raw identifier** when you have a raw identifier (URL, handle) and need to specify its entity and source context.

## How It Works

This tool queries the [Indexer's Materialized View](/docs/indexing#resolved-statements) to reveal the underlying graph structure.

1.  **Input Resolution**: Checks if the input is a known alias or primary ID.
2.  **Cluster Visualization**: Displays all other identifiers in the same [Identity Cluster](/docs/schema/relationships#sameas).
3.  **Primary Election**: Highlights the **Canonical Primary** (determined by `MIN(fingerprint)` of the cluster).
4.  **Trust Audit**: Shows the `owl:sameAs` links and their trust evaluations (`Fide-OwlSameAs-Person-v1`, etc.) that formed the cluster.

Enter a full Fide ID (e.g. `did:fide:0x15...`) in the first tab, or use the **Fide ID Parts** tab with entity type, source type, and raw identifier (e.g. `https://x.com/alice`) when you have a raw identifier.

## Understanding the Output

### Resolution Status

*   ✅ **Resolved**: The identifier is part of a trusted cluster.
*   ⚠️ **Unresolved**: The identifier exists but has no trusted `owl:sameAs` links.
*   ❌ **Unknown**: The identifier has never been attested to in the graph.

### Identity Cluster

The **Identity Cluster** shows all identifiers that are considered "the same entity" by the protocol. This includes:
*   **Canonical Primary**: The single Fide ID used for all protocol interactions.
*   **Aliases**: Other identifiers (e.g., social handles, DIDs) linked via trusted `owl:sameAs`.
*   **Trust Score**: The evaluation result that validated the link (e.g., `1` for Verified).

### Graph Provenance

Every link in the cluster is backed by a signed **Statement**. The Inspector allows you to inspect the raw statement data (Subject, Predicate, Object, Signature) to verify the source of the identity claim.
