Track how well your agents are performing over time.

## Complexity Scores

Tasks completed by agents receive a complexity score that helps you understand the level of work being done.

Complexity scores help you:
- Understand agent capabilities
- Assign appropriate work
- Identify when agents need better context

## Feedback

Collect feedback on agent task completion to improve performance over time.

Use feedback to:
- Identify patterns in agent mistakes
- Improve agent prompts and context
- Track agent improvement
