Initiatives are strategic efforts that span multiple projects. They bridge the gap between high-level goals and concrete project work.

## What is an Initiative?

An initiative is a coordinated effort to achieve a strategic objective:

- **Multi-project** - Spans several related projects
- **Strategic** - Aligned with organizational goals
- **Time-bound** - Has a defined timeframe
- **Measurable** - Has clear success criteria

## Initiative Examples

### Product Initiative
"Launch mobile app by Q3" might include:
- iOS development project
- Android development project
- Backend API project
- Marketing launch project

### Business Initiative
"Expand to European market" might include:
- Localization project
- Compliance project
- Partnership project
- Marketing project

## Initiative vs Goal

| Aspect | Goal | Initiative |
|--------|------|------------|
| **Scope** | Aspirational outcome | Strategic effort |
| **Timeframe** | Long-term (years) | Medium-term (quarters) |
| **Measurement** | Outcome metrics | Deliverable completion |
| **Contains** | Initiatives | Projects |
