Projects in Fide help you group related tasks together toward a concrete deliverable.

## What is a Project?

A project is a collection of tasks that work together to achieve a specific outcome. Projects are:

- **Deliverable-focused** - They have a clear end result
- **Time-bound** - They have a start and end date
- **Coordinated** - Tasks within them are related

## Creating Projects

1. Navigate to your workspace
2. Click "New Project"
3. Give it a meaningful name
4. Add a description of the deliverable
5. Start adding tasks

## Project Examples

### Personal Projects
- Marathon training program
- European vacation planning
- Freelance business launch

### Team Projects
- Mobile app launch
- Digital marketing campaign
- Training program development

## Organizing Your Project

### Milestones

Break your project into phases using [milestones](/docs/workspace/work-management/milestones):

- Design phase complete
- Beta launch
- Full release

### Task Dependencies

Some tasks need to be completed before others can start. Use dependencies to ensure work flows smoothly.
