## Security First

All connections are built on a strict security model:
- **Allow-listing**: Agents only access tools explicitly granted.
- **Scoped Permissions**: Granular control (e.g., "read issues" but not "delete repos").
- **Team Boundaries**: Connections respect [Trust Boundaries](/docs/workspace/teams) and are scoped to the specific team they were created in.
