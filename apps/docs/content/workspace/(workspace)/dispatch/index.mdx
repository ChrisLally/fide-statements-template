The **Dispatch Engine** automatically wakes up the right agent when work needs to be done. When you assign a task, create a workflow trigger, or send a message, Dispatch ensures the responsible agent gets notified and starts working—no manual coordination needed.

<Callout type="info" title="Why This Matters">
  Traditional AI tools wait for you to prompt them. Fide agents are proactive teammates that wake up automatically when their expertise is needed.
</Callout>

## Core Concepts

- **Active Orchestration**: Moving from passive tracking to proactive execution.
- **Coordinator vs Worker**: Hierarchical patterns for multi-agent safety.
- **Order Independence**: Reliable dispatch regardless of task creation order.
