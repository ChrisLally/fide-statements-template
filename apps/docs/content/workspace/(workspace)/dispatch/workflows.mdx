> **Note**: This page has moved. Workflows are a separate feature from the Dispatch system.

See [Automation](/docs/workspace/automation) for documentation on user-created automation tools.

## What's the Difference?

| | Dispatch | Workflows |
|---|---|---|
| **Purpose** | Wake agents when tasks assigned | User-created automation tools |
| **Trigger** | Automatic (task creation/completion) | On-demand (manual or agent-invoked) |
| **Created by** | System | Users via visual builder |
| **Use case** | Agent coordination | Repeatable automation |
